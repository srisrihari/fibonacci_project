Module: fibonacci.py
Description: Contains functions for calculating Fibonacci numbers.

- fib_recursive(n): Calculates the nth Fibonacci number recursively.
- fib_iterative(n): Calculates the nth Fibonacci number iteratively.

Usage:
- Import this module into your Python program to use the functions.
- Call fib_recursive(n) or fib_iterative(n) with the desired value of n to get the nth Fibonacci number.
